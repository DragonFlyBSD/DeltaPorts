# DPorts Python v2 (Archived)

The legacy Python `dports` v2 toolchain is retired from this repository.

Use `dportsv3` for active workflows:

- `dportsv3 compose`
- `dportsv3 migrate ...`
- `dportsv3 dsl ...`

Legacy shell scripts under `scripts/generator/*.sh` are still present for
parity and operational reference.
